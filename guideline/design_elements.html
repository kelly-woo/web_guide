Font

잔디 서비스 내에서 사용하는 서체는 아래와 같으며 공통으로 쓰이는 영어서체를 위해 'Roboto'와 'Arial'을 먼저 선언합니다.

한국어-맑은 고딕 / Noto Sans Korean
{ font-family:'Roboto', Arial, 'Malgun Gothic', '맑은 고딕' , 'Noto Sans Korean', sans-serif, serif }
English-Roboto / Arial
{ font-family:font-family: 'Roboto', Arial, sans-serif, serif }
日本語-"Hiragino Kaku Gothic Pro" / "ヒラギノ角ゴ Pro W3",
{ font-family:'Roboto', Arial, "Hiragino Kaku Gothic Pro", "ヒラギノ角ゴ Pro W3", "メイリオ", helvetica, sans-serif }
简体中文
{ font-family:'Roboto', Arial, "Helvetica Neue",Helvetica,Arial,sans-serif }
繁體中文
{ font-family:'Roboto', Arial, "Helvetica Neue",Helvetica,Arial,sans-serif }

Layout

App

Icons
